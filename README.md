# ☕ CoffeeApp Showcase

**CoffeeApp** is a production-ready Android application available on Google Play, designed to deliver a modern and user-friendly coffee ordering experience. The application enables users to explore coffee and dessert menus, place orders, manage balances, and track past purchases within a clean and responsive Material Design interface.

**Application Name:** **CoffeeApp**  
**Google Play:** **[View on Google Play](https://play.google.com/store/apps/details?id=com.burakalmaz.coffeeapp)**

---

## Overview

Developed with a focus on usability and responsive mobile design, CoffeeApp offers a practical ordering workflow supported by Firebase Authentication and Realtime Database. The project demonstrates modern Android development practices while providing a polished user experience for menu browsing, cart handling, and account management.

---

## Tech Stack

- **Kotlin**
- **Android Jetpack** (ViewModel, LiveData, Navigation)
- **Material Components**
- **RecyclerView**
- **ViewBinding**
- **Firebase Authentication**
- **Firebase Realtime Database**

---

## Key Features

- Firebase-based user authentication
- Coffee and dessert menu browsing
- Cart management with size and quantity selection
- Balance deposit and spending functionality
- Order history tracking with detailed timestamps
- Smooth interface supported by animations and dialogs

---

## What This Project Demonstrates

- Android application development with Kotlin
- Firebase integration for authentication and real-time data operations
- Clean and responsive Material UI design
